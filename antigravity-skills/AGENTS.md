## SISTEMA DE SQUADS ATIVO

Este projeto usa o **CEO Orchestrator** com 12 squads especializados. Para ativar, use:

```
@ceo-orchestrator novo projeto: [descreva seu projeto aqui]
```

ou em projetos existentes:

```
@ceo-orchestrator analise este projeto e ative os squads relevantes
```

## SQUADS DISPONÍVEIS

Cada squad é uma skill global em `~/.gemini/antigravity/skills/`. Você pode ativar diretamente:

| Comando | Squad | Especialidade |
|---------|-------|---------------|
| `@ceo-orchestrator` | CEO | Orquestra todos os squads |
| `@c-level-squad` | C-Level | Estratégia, visão, C-suite |
| `@brand-squad` | Brand | Marca, naming, identidade |
| `@copy-squad` | Copy | Copywriting, vendas, emails |
| `@design-squad` | Design | UI/UX, design systems |
| `@data-squad` | Data | Analytics, growth, retenção |
| `@traffic-masters` | Traffic | Tráfego pago, mídia |
| `@hormozi-squad` | Hormozi | Oferta, leads, vendas |
| `@storytelling` | Story | Narrativa, pitch, conteúdo |
| `@advisory-board` | Advisory | Conselho estratégico |
| `@movement` | Movement | Comunidade, movimento |
| `@cybersecurity` | Security | Segurança, pentest |
| `@claude-code-mastery` | Code | Claude Code, MCP, automação |
